# PortScript*
### *Node.js&copy; with fs is required

<br/>
Interpretador de scripts em português para Javascript com propositos educacionais...

const -> constante

<br/>
false-> falso
<br/>

<br/>

<br/>
function-> funcao
<br/>

<br/>
switch-> troque
<br/>

<br/>

.forEach()-> .paraCada() (Array function)

.substring()-> .fragmento() (String function)

<br/>
