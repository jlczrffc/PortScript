# PortScript*
### *Nodejs with fs required
<br/>
Portuguese script interpreter for educational purpose...
<br/>
<br/>

<br/>
<br/>
for   -> para
<br/>
let   -> permita
<br/>
if    -> se
<br/>
const -> constante
<br/>
true  -> verdadeiro
<br/>
false -> falso
<br/>
do    -> faca
<br/>
while -> enquanto
<br/>
function -> funcao
